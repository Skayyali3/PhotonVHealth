## How to Run:

### 1. Clone the repository and enter:
```bash
git clone https://github.com/Skayyali3/REPONAME.git
cd REPONAME
```
### 2. Make a virtual environment:
```bash
python -m venv venv
source venv/bin/activate # Linux/macOS
venv\Scripts\activate
```

### 3. Install requirements:
```bash
pip install -r requirements.txt
```

### 4. Run the Flask app:
```bash
python main.py
# Open your browser to http://127.0.0.1:5000 to check out the site locally!
```
